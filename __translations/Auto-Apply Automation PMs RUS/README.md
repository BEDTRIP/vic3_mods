[h1] Оригинальное описание [/h1]
Этот мод можно установить в середине игры.
Все функции можно включать и отключать в любой момент — они затрагивают только игрока.
Мод включает только автоматизационные производственные методы (зелёные ПМ).
Для других типов PM используйте мод: 
[url=https://steamcommunity.com/sharedfiles/filedetails/?id=3519368804] Auto-Apply PMs RUS [/url].
[h2] Автоматическое применение автоматизации ПМ (Auto-Apply Automation PMs) [/h2]
[list]
[*]Постепенно активирует производственные методы автоматизации в областях с нехваткой рабочей силы.

[*]Авто-применение можно включать и отключать отдельно для разных типов автоматизации:
    - Базовая автоматизация → Электрификация → Нефтяная автоматизация
    - Железнодорожный транспорт

[*]Изменение PM зависит от местных цен, например (PM1 → PM2):
    - PM1 требует 15 инструментов и 10 угля
    - PM2 требует 10 инструментов и 20 угля
    - Проверяется только цена угля, так как инструментов PM2 требует меньше

[*]Автоматизация начинается с самых низкоуровневых зданий среди всех подходящих зданий в области, чтобы процесс проходил плавно и не вызывал резких колебаний цен на местном рынке.
[/list]
[h2] Как пользоваться [/h2]

[*]Используйте решение «Управление производственной автоматизацией», чтобы активировать запись в журнале
[/list]
